# 什么是 Vue DevTools Unlocker？

Vue DevTools Unlocker 是一个浏览器扩展，用于在生产环境页面中启用 Vue DevTools 面板。

在生产构建中，Vue 默认会关闭 DevTools。这个扩展会向页面注入一小段脚本，重新启用 Vue 的 devtools hook，让你的 Vue 应用在生产环境也能像开发环境一样被检查和调试。

## 主要特性

* 在生产环境页面解锁 Vue DevTools 面板。
* 同时支持 Vue 2 和 Vue 3。
* 在扩展弹窗中显示检测到的 Vue 版本。
* 无需修改应用源码或构建配置。
* 基于浏览器扩展的调试流程，贴合现有开发习惯。
