Vue DevTools Unlocker在生产环境启用 Vue DevTools

一个浏览器扩展，在生产站点解锁 Vue DevTools，支持 Vue 2 和 Vue 3。

开始使用 👆
查看 GitHub

在生产环境解锁 DevTools

在生产环境页面启用 Vue 面板，无需修改应用源码。

同时支持 Vue 2 与 Vue 3

自动检测 Vue 版本，为两个主版本解锁调试能力。

显示 Vue 版本信息

在扩展中展示检测到的 Vue.js 版本，方便确认线上运行环境。

查看 Pinia 状态

当页面存在运行时 Pinia 实例时，可查看 Vue 3 Pinia stores 与 state。
