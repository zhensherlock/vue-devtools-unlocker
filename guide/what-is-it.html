What is Vue DevTools Unlocker? ​

Vue DevTools Unlocker is a browser extension that enables the Vue DevTools panel on production sites.

By default, Vue DevTools is disabled in production builds. This extension injects a small script into the page to re-enable the Vue devtools hook so that your Vue app can be inspected just like in development.

Key Features ​

  • Unlocks the Vue DevTools panel on production pages.
  • Works with both Vue 2 and Vue 3.
  • Shows the detected Vue version in the extension popup.
  • No changes required to your application code or build config.
  • Simple browser extension workflow that fits into existing debugging habits.

Supported Browsers ​

Vue DevTools Unlocker is available in the following browser stores:

  • Chrome Web Store
  • Microsoft Edge Add-ons
  • Firefox Add-ons
